##OP.GG * opgg是一家专注于做LOL数据的网站,其数据来源主要为Riot开放的API,主营KR有NA EU支持 * 有搜索ID功能 * 有排行榜 * 有ID战绩列表,符文天赋 * 分析ID的数据,英雄,位置 * 高分段录像 ##Riot API * Riot API是Riot提供的一套官方的数据接口,只针对Riot直属的服务器,所以国服,台服等不在此列 * https://developer.riotgames.com 拥有一个对应服务器的账号,登录以后就可以获得一个development key了 * 如果需要专业级别的key,可以申请 * MatchHistory * MH是新出的战绩服务器,比原先的战绩更加详细,但只能获取排位数据 * MH拥有Timeline功能,可以记录更多比赛中的数据,更加详细便于分析 * 大型比赛也有MH,一般公布在LOLEsports上,有一套自由的API,以后详述 * ddragon * 官方的CDN服务器 * 保存了一些LOL的静态资料,包括文字,图片等 * http://ddragon.leagueoflegends.com/tool 有一个在线工具 * API部分以后详述 ##LOL Replay * op.gg的replay是自行录制的非官方提供,方法在github上有很多开源项目,原理是借助了Riot的OB系统,可以通过抓包过程来学习 * Riot API可以提供录制replay需要的参数 * 播放方式是自建一个http服务来模拟OB服务器的动作 ##TGP录像 * 在之前的TGP中也有一个录像,原理类似 * 只不过他们把数据打包成一个文件,在玩家本地自建了http服务实现的 ##国服LOL录像 * 国外一些录像工具在国服没法使用是因为国服有TP阻止了这些软件获得客户端数据导致的 * 可以尝试使用lolreplay的一个库,通过抓包的方式把客户端数据抓包并解析来取得需要的数据,那个库叫air.dll

OP.GG

  • opgg是一家专注于做LOL数据的网站,其数据来源主要为Riot开放的API,主营KR有NA EU支持
    • 有搜索ID功能
    • 有排行榜
    • 有ID战绩列表,符文天赋
    • 分析ID的数据,英雄,位置
    • 高分段录像

Riot API

  • Riot API是Riot提供的一套官方的数据接口,只针对Riot直属的服务器,所以国服,台服等不在此列
    • https://developer.riotgames.com 拥有一个对应服务器的账号,登录以后就可以获得一个development key了
    • 如果需要专业级别的key,可以申请
  • MatchHistory
    • MH是新出的战绩服务器,比原先的战绩更加详细,但只能获取排位数据
    • MH拥有Timeline功能,可以记录更多比赛中的数据,更加详细便于分析
    • 大型比赛也有MH,一般公布在LOLEsports上,有一套自由的API,以后详述
  • ddragon
    • 官方的CDN服务器
    • 保存了一些LOL的静态资料,包括文字,图片等
    • http://ddragon.leagueoflegends.com/tool 有一个在线工具
    • API部分以后详述

LOL Replay

  • op.gg的replay是自行录制的非官方提供,方法在github上有很多开源项目,原理是借助了Riot的OB系统,可以通过抓包过程来学习
  • Riot API可以提供录制replay需要的参数
  • 播放方式是自建一个http服务来模拟OB服务器的动作

TGP录像

  • 在之前的TGP中也有一个录像,原理类似
  • 只不过他们把数据打包成一个文件,在玩家本地自建了http服务实现的

国服LOL录像

  • 国外一些录像工具在国服没法使用是因为国服有TP阻止了这些软件获得客户端数据导致的
  • 可以尝试使用lolreplay的一个库,通过抓包的方式把客户端数据抓包并解析来取得需要的数据,那个库叫air.dll
